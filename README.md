## üöÄ Getting Started

### Prerequisites

- Python 3.9+
- An OpenAI (or compatible LLM provider) API key
- [LangGraph](https://pypi.org/project/langgraph/) and LangChain installed

### Installation

```bash
# Clone the repository
git clone https://github.com/somrajmondal/agentic_workflow_using_langraph.git
cd agentic_workflow_using_langraph

# Create and activate a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Environment Setup

```bash
# Copy the env template and fill in your credentials
cp ".env copy" .env
```

Edit `.env` and add your API keys:

```env
OPENAI_API_KEY=your_openai_api_key_here
# Add any other required keys
```

## üßÝ How It Works

This project uses **LangGraph** to build a directed graph of agents where:

- **`state.py`** defines the shared `AgentState` passed between nodes
- **`workflow.py`** assembles the graph ‚Äî registering nodes (agents) and defining edges (routing logic)
- **`agents/`** contains individual agent logic (e.g., researcher, writer, reviewer)
- **`config.py`** centralizes model and environment configuration
- **`main.py`** compiles and runs the graph

```
[START] ‚Üí Agent A ‚Üí Agent B ‚Üí ... ‚Üí [END]
               ‚Üë         ‚Üì
            (conditional routing via state)
```
